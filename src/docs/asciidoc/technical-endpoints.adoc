This page documents the technical endpoints exposed outside the business controllers.

These paths are intended for trusted internal or devops access rather than the supported external `/api/**` contract. `GET /actuator/prometheus` remains supported for trusted deployment scraping, but it is deployment-scoped rather than part of the internet-public contract.

Redirects to the generated HTML documentation bundled with the application for trusted internal review and runtime validation.

== `GET /actuator/prometheus`

Returns Prometheus-formatted metrics for scraping. This includes the default Spring and JVM meters, cache activity meters for the in-memory caches, and custom application meters under the `technical.interview.demo.*` prefix.

This endpoint is intended for trusted deployment scraping such as the in-cluster ServiceMonitor assets checked into this repository. It is supported for that purpose, but it is not part of the internet-public endpoint contract.
